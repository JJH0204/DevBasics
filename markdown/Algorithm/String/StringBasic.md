# strlen()

|        |                                        |
| :----- | :------------------------------------- |
| 헤더   | #include <string.h>                    |
| 형식   | size_t strlen(const char \*s);         |
| 기능   | s가 가리키는 문자열의 길이를 반환한다. |
| 반환값 | 구한 문자열의 길이 반환                |

# strchr()
|        |                                        |
| :----- | :------------------------------------- |
| 헤더   | #include <string.h>                    |
| 형식   | size_t strchr(const char \*s, int c);         |
| 기능   | s가 가리키는 문자열에서 가장 앞쪽에 있는 c를 찾는다.<br/>이때 c는 널 문자여도 가능 |
| 반환값 | 찾은 문자에 대한 포인터를 반환<br/>문자가 없으면 널 포인터 반환                |
